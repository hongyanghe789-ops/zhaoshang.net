## 一、功能概述

### 1.1 背景与目标

大型园区运营商/城投/国企通常管理多个产业园区，需要在集团层面进行统一管控、数据汇总、跨园区协作。本功能提供**集团级管控能力**，包括集团看板、统一租客库、集团级权限、统一配置、排名激励，以及导出功能优化和园区资料快速发送。

### 1.2 核心价值

- 集团总部一屏掌握所有园区经营数据
- 统一租客库，避免同一企业在不同园区重复管理
- 集团级ABAC权限，灵活控制跨园区数据访问
- 园区间排名激励，促进良性竞争

## 二、功能模块

### 2.1 集团看板

**入口**：工作台 → 集团看板（仅集团管理员角色可见）

**看板内容**：

| 看板区域 | 指标 |
|---------|------|
| 总览卡片 | 管理园区总数、总管理面积、总出租率、总签约合同数、总应收/实收 |
| 园区排行 | 出租率排行、签约面积排行、收入排行、招商转化率排行 |
| 趋势图表 | 集团出租率月度趋势、集团收入月度趋势 |
| 园区对比 | 多园区核心指标横向对比（雷达图/柱状图） |
| 预警汇总 | 各园区合同到期预警、账单逾期预警、水电异常预警 |

### 2.2 统一租客库

| 功能 | 说明 |
|------|------|
| 集团租客视图 | 跨园区汇总同一企业的所有合同/房源/账单 |
| 企业去重 | 基于统一社会信用代码自动识别同一企业 |
| 集团客商画像 | 汇总企业在所有园区的行为数据 |
| 跨园区推荐 | 企业在A园区退租 → 推荐B园区合适房源 |

### 2.3 集团级权限（ABAC联动）

基于ABAC模型扩展集团管控策略：

| 策略 | 规则 | 效果 |
|------|------|------|
| 集团全局可见 | Subject.角色标签 CONTAINS "集团管理员" | 所有园区数据可见 |
| 多园区管辖 | Subject.管辖园区 CONTAINS Resource.所属园区 | 仅管辖园区可见 |
| 跨园区只读 | Subject.角色标签 CONTAINS "集团巡视员" | 全局可见但只读 |
| 敏感数据保护 | Subject.职级 < "总监" AND Resource.数据密级 == "高" | 字段脱敏 |

### 2.4 统一配置

| 配置项 | 说明 |
|--------|------|
| 集团统一模板 | 合同模板/审批流程/计费规则可从集团下发到各园区 |
| 园区个性化 | 园区可在集团模板基础上进行个性化调整（需审批） |
| 配置同步 | 集团修改配置后，选择推送范围（全部园区/指定园区） |
| 版本管理 | 配置变更历史记录，支持回滚 |

### 2.5 排名激励

| 排名维度 | 周期 | 展示 |
|---------|------|------|
| 出租率 | 月度/季度/年度 | 排行榜 + 变化趋势 |
| 招商签约面积 | 月度/季度/年度 | 排行榜 + 目标完成率 |
| 收入完成率 | 月度/季度/年度 | 排行榜 + 同比增长 |
| 租客满意度 | 季度 | 排行榜 + 评分分布 |

## 三、导出功能优化

### 3.1 导出选择弹窗

所有列表导出操作统一改为弹窗确认模式：

```
┌────────────────────────────┐
│      导出数据                │
│                            │
│  ○ 导出当前筛选结果 (128条)  │
│  ○ 导出全部数据 (5,234条)   │
│                            │
│  [取消]         [确认导出]  │
└────────────────────────────┘
```

### 3.2 文件命名规则

格式：`{模块名}_{导出日期}_{筛选条件摘要}.xlsx`

示例：`合同列表_20260227_创新园区_执行中.xlsx`

### 3.3 大数据量异步导出

- 超过5000条 → 自动转为异步导出
- 异步导出任务进入"传输记录"
- 完成后通过消息通知中心推送下载链接

## 四、园区资料快速发送

### 4.1 功能入口

- 园区详情页 → "发送资料"按钮
- 建筑详情页 → "发送资料"按钮
- 客商详情页 → "发送资料"按钮（选择要发送的园区/建筑）

### 4.2 操作流程

1. 点击"发送资料"
2. 选择接收人（从客商联系人中选择，或手动输入手机号/邮箱）
3. 选择发送内容（园区介绍/建筑介绍/房源列表/配套设施/政策优惠）
4. 选择发送方式（IM消息/短信/邮件）
5. 确认发送

### 4.3 资料内容

| 资料类型 | 内容 | 格式 |
|---------|------|------|
| 园区介绍 | 园区基础信息+配套+交通+政策 | 图文卡片/PDF |
| 建筑介绍 | 建筑信息+楼层平面图+可用房源 | 图文卡片/PDF |
| 房源推荐 | 匹配客商需求的房源列表 | 图文卡片 |
| 政策优惠 | 当前适用的优惠政策 | 图文卡片 |

## 五、数据权限

- 集团看板：集团管理员/集团只读角色
- 统一租客库：集团管理员可查看全部，园区管理员仅看本园区
- 统一配置：集团管理员可下发，园区管理员可调整本园区
- 资料发送：有客商管理权限的用户
- 导出功能：受ABAC策略控制

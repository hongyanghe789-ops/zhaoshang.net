## 一、功能概述

### 1.1 背景与目标

现有剖面图提供4种颜色视图（出租状态视图），但缺乏市场数据维度的可视化能力。本功能新增**"市场热度"视图模式**，基于招商网络平台的流量数据用热力图渲染房源关注度；同时引入**周边竞品均价参考**，辅助定价决策。

### 1.2 核心价值

- 直观看到哪些房源最受市场关注，辅助招商策略调整
- 基于竞品数据辅助定价，避免定价过高或过低
- 数据驱动的载体运营决策

## 二、市场热度视图

### 2.1 视图切换

**入口**：剖面图页面 → 顶部视图切换下拉框

原有视图：
- 出租状态视图（默认，4种颜色）

新增视图：
- **市场热度视图**（热力图渲染）

### 2.2 热力图渲染规则

基于招商网络平台回传的流量数据（参考载体发布反馈数据.md），计算每个房源的"热度分值"：

**热度分值计算公式**：
```
热度分值 = PV权重(0.1) × 页面浏览量
          + UV权重(0.15) × 独立访客数
          + 收藏权重(0.2) × 收藏量
          + IM权重(0.25) × IM咨询数
          + 电话权重(0.3) × 电话拨打数
```

**颜色映射**：

| 热度等级 | 分值区间 | 颜色 | 说明 |
|---------|---------|------|------|
| 极热 | ≥ 80分 | 深红色 #FF1744 | 高度关注，建议优先跟进 |
| 较热 | 60~79分 | 橙红色 #FF6D00 | 关注度较高 |
| 一般 | 40~59分 | 橙色 #FFA000 | 关注度中等 |
| 较冷 | 20~39分 | 浅蓝色 #42A5F5 | 关注度偏低，建议优化标签/图片 |
| 冷门 | < 20分 | 深蓝色 #1565C0 | 极少关注，建议调整定价/重新包装 |
| 未发布 | 无数据 | 灰色 #BDBDBD | 尚未发布到平台 |

### 2.3 热度详情弹窗

点击热力图中的房源块，弹出详情：

```

```

### 2.4 热度时间范围

支持切换统计时间范围：近7天 / 近30天（默认）/ 近90天

## 三、周边竞品均价参考

### 3.1 数据来源

引入招商网络平台宏观数据API，获取同城市/同区域/同类型载体的均价数据。

### 3.2 展示位置

| 展示位置 | 形式 |
|---------|------|
| 房源编辑页 → 价格字段旁 | 灰色提示文字："周边均价 ¥X.XX/㎡·月" |
| 载体发布页 → 价格确认区 | 对比提示："当前定价高于/低于周边均价 X%" |
| 剖面图热度视图 → 房源弹窗 | 价格对比指标 |

### 3.3 偏离提示规则

| 偏离幅度 | 提示颜色 | 提示文案 |
|---------|---------|---------|
| 高于均价 > 30% | 红色 | "当前定价显著高于周边均价，可能影响去化速度" |
| 高于均价 10%~30% | 橙色 | "当前定价略高于周边均价" |
| 均价 ±10% | 绿色 | "当前定价与周边均价持平" |
| 低于均价 10%~30% | 蓝色 | "当前定价低于周边均价，可考虑适当上调" |
| 低于均价 > 30% | 紫色 | "当前定价显著低于周边均价" |

### 3.4 竞品均价统计维度

| 维度 | 说明 |
|------|------|
| 同城均价 | 同城市所有同类型载体均价 |
| 同区域均价 | 同行政区/商圈的载体均价 |
| 同类型均价 | 同载体类型（写字楼/厂房/仓库等）均价 |
